API Project: Timestamp Microservice


The purpose of this page is to serve the testing functioanlity of API endpoint: GET [project_url]/api/timestamp/:date_string?



Example Usage:


Valid date_string Milliseconds date_string Empty date_string Invalid date_string


Example URL: [project url]/api/timestamp/2015-12-25

Example Output: {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}



Example URL: [project url]/api/timestamp/1450137600

Example Output: {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}



Example URL: [project url]/api/timestamp/

Example Output: {"unix":1540660589902,"utc":"Sat, 27 Oct 2018 17:16:29 GMT"} (As of 2018-10-27)



Example URL: [project url]/api/timestamp/typeAnythingInvalid

Example Output: {"error" : "Invalid Date" }
